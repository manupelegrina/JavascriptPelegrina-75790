Bienvenido a Hardware Online

Este simulador permite agregar productos a un carrito de compras mediante la consola de JavaScript.

¿Cómo funciona?

1. En la consola de JavaScript, se mostrarán los productos disponibles.
2. Debes ingresar el ID del producto que deseas comprar.
3. Puedes seguir agregando productos hasta que decidas finalizar.
4. Al terminar, verás un resumen con el total de la compra.
